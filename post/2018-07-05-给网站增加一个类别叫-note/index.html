文章 2018-07-05

给网站增加一个类别叫 note

话说接触 Python 这货也很久了，不过一段时间后就等着跨一个坎，不然对技术的理解如何升阶。

要说有哪本书完全啃下来了，肯定地说，一本都没有，真是情况是，大多是泛泛而看，等到下一段时间，又被新的信息所吸引，这就是信息焦虑，如果无法处理得好的话，最终就是什么东西都是只是听到过见到过甚至开始过，但是，却并没有深入过。

时间过去了，而你，最终什么都不擅长。

倒不是说绝对的不好，见识广阔思路开放，是有必要的，但也总要选一个入口走深一点，知道了全国几百个溶洞的所在地，可一个都没真正走进去过的话，这就不能叫溶洞探险家。

我时常会望着架子上的书脊发呆，望着那些名字的背后有着多么深奥的学问，也带有一种难以诉说的恐惧感，就如同乘一叶小舟漂浮在大海之上，可技术海洋太大又太深，每日还波涛汹涌不断快速进化，恐怕是处于这个时代的 IT 技术人实实在在的切身感受。

但又说时代更好也罢，知识付费拔地而起，原来难以寻找的知识现在人家双手捧上，而唯独永远欠缺的，是 shi jian，是时间，也是实践。

不求太多，只求一点一滴吧，何况生活情绪万变，儿女情长，家长里短各种事务，人们永远怀念学生时代，这都是有切实支撑的。

增加的 note 板块暂时先只在 Jeremy’s blog 进行更新，往后是否发公众号再看情况，一些碎碎片片的东西，算是找个地方让它们留下点记忆。
