Saleor 22: 架构 - 缩略图

Saleor 使用 VersatileImageField 替换了 Django 的 ImageField 。出于性能原因，在非调试模式下，缩略图由工作者的任务系统预先生成，在保存实例后触发。访问丢失的图像将导致404错误。

在调试模式下，按需生成缩略图。

手动生成产品缩略图

为所有 ProductImage 实例创建缺失的缩略图。

$ python manage.py create_thumbnails

删除图像

Image 实例不会自动删除图像格式，主图像也不会自动删除。有关删除图像的更多信息，请参见 VersatileImageField 文档
